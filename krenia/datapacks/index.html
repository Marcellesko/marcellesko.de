Krenia Datapacks

General Information:

Some features seen on the Krenia Server aren't contained in vanilla Minecraft (the base game). Most of the small changes are achieved with so called datapacks, that use vanilla Minecraft commands to create new functionalities like changed or added crafting recipes, enhanced trades or adapted mob and block behavior.
The first list shows all the installed datapacks, what they do, how to use them and also links to the original download sites.

Some other functions on the server like the ingame voice chat aren't possible just with datapacks, but with mods. Those Mods can change a whole lot more of the game and add completely new functions.
The second list shows all the mods that are installed on the server.

Recipe Changes:

There are a lot of new or changed crafting recipes that make some quality-of-life changes. You can access the full list of recipe additions and changes by clicking on the i.
Edited to fit usage.

Datapack List:

|NEW| - Bell Finds All Raiders:

The limited radius of bells giving raiders the glowing effect has been removed. Now every loaded Raider will receive the effect after a bell has been rung.

|NEW| - End Crystal Bossbar:

Now the Ender Dragon fight has another bossbar, indicating how many End Crystals are left.

|NEW| - Item Despawn Particles:

When items get close to despawning, they now briefly emit particles to show that they are about to disappear.

|NEW| - Name Tag Drops:

When a mob has been named with a name tag, it will now drop that name tag when it dies.

|NEW| - Visible Shulker Boxes Names:

When a shulker box has been renamed it will now have a name tag with its name above when placed down.

AFK Display:

When a player stands completely still for 5 minutes the name will turn to a light gray. When that player stands still for a total of 30 minutes the name will go from light gray to a dark gray. When moving around again, the name will turn back to white.
Edited to fit usage.

Anti Enderman Griefing:

Endermen will no longer be able to pick up blocks.

Armor Statues:

With the use of a special book, armor stands can be positioned and customized in great detail. Their seperate parts can be rotated individually, the whole stand can be resized, moved, locked and can be made invisible. Item frames can be locked and made invisible.

Custom Nether Portals:

Nether portals are no longer limited to a rectangle and can now be shaped freely.

Custom Player Heads:

Using a special crafting table, bought from a mason villager, all player heads can now be turned into custom heads with the help of the Minecraft Heads website.
Edited to fit usage.

Double Shulker Shells:

Shulkers now drop two shells instead of only one.

End Above Overworld:

The end is now seemingly located on top of the overworld. When flying above the y-coordinate of 400 the player will get teleported to the end and when falling into the void, the player will get teleported to the sky of the overworld.
Self Made.

Fast Leaf Decay:

When a tree trunk is removed the leaves will no longer disappear by random tick but quickly right after removing the log.

Player Head Drops:

When a player is killed by another player, their head will now drop.

Regional Wandering Traders:

Wandering Traders now have a wider trade offer that is dependant on the type of trader. There are new regional varieties.

Silenced Mobs:

When renaming a mob to "silenced" they will no longer make any sounds and can be unmuted by naming them "unsilence". By renaming any mob "baby" they will permanently be a baby varient of that mob (if they exist) and by naming it "unbaby" it will start to grow up again.
Edited to fit usage.

Wandering Trader Notification:

When a wandering trader spawns, a chat notifying message will be sent to everyone, stating which player is the closest to it.


Mod List:

AI Improvements:

With this mod the AI behavoir of mobs is improved, making an impact on the performance of the server but not really affecting the game mechanics, that players would notice.

Dynamic View:

This mod adapts the server render and simulation distance to the amount of players that are online. This allows the server to save resources when a lot of people are online and still make the game look enjoyable when only a few are playing.

Lithium:

This is a performance enhancing mod that drasticly improves tickrates and makes the server run smoother.

Simple Voice Chat Mod:

This mod allows the players that have it installed on ther client to talk via voice chat directly on the server.

Too fast:

When a server is overloaded and someone for example, flies with an elytra, rubberbanding often occurs and the player gets lagged back. This mod gets rid of that effect.

No Chat Reports:

Because we manage issues internally on Krenia and therefore don't need the new report system, we have this mod that gets rid of a few annoying things that feature brings along.
